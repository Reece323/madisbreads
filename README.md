# 🍞 Madi's Breads

A beautiful, modern recipe website for homemade bread making — featuring protein breads, healthy wholesome loaves, old-time grandma recipes, and more.

**Live site:** [madisbreads](https://YOUR_USERNAME.github.io/madisbreads/)

## Features

- **25+ bread recipes** with full ingredients, step-by-step instructions, and baking tips
- **5 categories** — Classic, Grandma's Kitchen, Protein, Healthy, Sweet Breads
- **Dark mode** toggle with saved preference
- **Ninja Creami page** — coming soon section for protein frozen treats
- **Fully responsive** — mobile-first design that looks great on any device
- **Recipe modals** — click any card to view the full recipe
- **Category filtering** — browse by bread type
- **Modern design** — olive green palette with cottage-core accents

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Main bread recipes site |

## Tech

- Pure HTML, CSS, and vanilla JavaScript
- No build tools or dependencies required
- Google Fonts: Libre Caslon Display, Outfit, Newsreader
- Hosted on GitHub Pages

## Local Development

Just open `index.html` in your browser — no server needed.

## Deployment

This site is deployed via **GitHub Pages** from the `main` branch root.

To set up:
1. Go to your repo **Settings → Pages**
2. Under **Source**, select `Deploy from a branch`
3. Choose `main` branch and `/ (root)` folder
4. Click **Save**

Your site will be live at `https://YOUR_USERNAME.github.io/madisbreads/`
